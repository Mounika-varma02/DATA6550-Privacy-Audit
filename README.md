# DATA6550 Privacy Audit Project

## Overview
This repository contains our group project for the Data Ethics course. The project focuses on analyzing privacy policies of various digital services and evaluating their compliance with data protection regulations such as GDPR and CCPA.

## Objectives
- Understand how services collect and use user data  
- Evaluate data sharing practices  
- Analyze user privacy rights and controls  
- Compare compliance with GDPR and CCPA  

## Repository Structure

### Agreements
Contains privacy policy documents collected for different services.

### Analysis
Contains individual analysis completed by each group member.

### Comparison
Contains comparison of privacy practices across selected services, including findings and regulatory analysis.

### Collaboration
Contains group work and progress documents.

## Key Findings
Most services collect user data to provide personalized features. While many platforms follow GDPR and CCPA by offering user rights such as access and deletion, challenges still exist in terms of transparency, data usage, and user awareness.
